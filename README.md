# Teslo API

1. Clone proyect
2. ```npm i```
3. Clone file ```.env.template``` and rename as ```.env```
4. Change the env variables values
5. Dockerize the database
```
docker compose up -d
```
6. Start proyect
```
npm run start:dev
```

7. Execute seeds
```
localhost:3000/api/seeds
```
