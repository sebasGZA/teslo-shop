# Teslo API

1. Clone proyect
2. Run ```npm install```

3. Clone file ```.template.env``` and rename to ```.env```

4. Set .env variables

5. Set database with ```docker-compose up -d```

6. Execute seeds
```
http://localhost:3000/api/seeds
```

7. Run dev: ```npm run start:dev```
